About This Project

This is a Love2D game project written in Lua. Love2D is a framework for making 2D games in Lua.

Project Files:

  • main.lua - Main game logic
  • conf.lua - Game configuration
  • README.md - Project documentation

How to Run This Game

To run this Love2D game locally on your computer:

  1. Install Love2D from love2d.org
  2. Download or clone this project
  3. Open a terminal in the project directory
  4. Run the command:
love .

Alternative Methods

You can also:

  • Drag the project folder onto the Love2D executable
  • Create a .love file by zipping the project contents

Note

This project cannot be run directly in a web browser as it requires the Love2D runtime environment.
